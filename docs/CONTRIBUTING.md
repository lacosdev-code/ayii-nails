# ü§ù Contributing to Ayii Nails

Thank you for your interest in contributing to the Ayii Nails project! We welcome contributions to improve the codebase and user experience.

## Getting Started

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally:
    ```bash
    git clone https://github.com/your-username/ayii-nails.git
    ```
3.  **Install dependencies**:
    ```bash
    npm install
    ```
4.  **Run the development server**:
    ```bash
    npm run dev
    ```

## Development Workflow

1.  **Create a branch** for your feature or fix:
    ```bash
    git checkout -b feature/amazing-feature
    ```
2.  **Make your changes**. Ensure you follow the existing code style (Prettier/ESLint).
3.  **Commit your changes** with clear, descriptive messages:
    ```bash
    git commit -m "feat: Add new booking modal"
    ```
4.  **Push to your fork**:
    ```bash
    git push origin feature/amazing-feature
    ```
5.  **Open a Pull Request** against the `main` branch of the original repository.

## Coding Standards

- **React**: Use Functional Components and Hooks.
- **Styling**: Use Tailwind CSS utility classes. Avoid inline styles.
- **Icons**: Use `react-icons` (Ionicons 5 preferred).
- **Images**: Optimize images before adding them to `src/assets`.

## Reporting Issues

If you find a bug or have a feature request, please open an issue in the repository with details on how to reproduce it.
